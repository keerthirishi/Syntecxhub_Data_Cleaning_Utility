# Syntecxhub_Data_Cleaning_Utility

A Python-based data cleaning utility developed as part of the Syntecxhub Data Science Internship.

Features

* Handles missing values
* Removes duplicate records
* Standardizes column names
* Converts date formats
* Generates a cleaned dataset
* Creates a cleaning log

Technologies Used

* Python
* Pandas

Run

```bash
python data_cleaning.py
```
